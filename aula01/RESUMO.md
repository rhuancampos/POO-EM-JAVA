# Aula 01: Introdução ao Java e Orientação a Objetos

Bem-vindo à sua primeira aula de Java! Nesta sessão, exploramos os fundamentos que tornam o Java uma das linguagens mais robustas e utilizadas no mundo corporativo.

## O que aprendemos hoje?

1.  **Ecossistema Java**:
    * **JDK (Java Development Kit)**: Nosso kit de ferramentas para criar código.
    * **JRE (Java Runtime Environment)**: O ambiente necessário para rodar o código.
    * **JVM (Java Virtual Machine)**: O motor que traduz o Java para qualquer sistema (Lema: *Write Once, Run Anywhere*).

2.  **Tipagem Estática**:
    * Diferente de linguagens dinâmicas, no Java precisamos declarar o tipo de cada variável (`int`, `String`, `float`, etc.).
    * **Primitivos vs Wrappers**: Vimos que `int` é um valor simples, enquanto `Integer` é um objeto com superpoderes.

3.  **Entrada de Dados**:
    * Uso da classe `Scanner` para interagir com o usuário via console.

4.  **Coleções (Lists)**:
    * Introdução ao `ArrayList`, uma forma dinâmica de armazenar dados que cresce conforme a necessidade, ao contrário dos Arrays fixos.

## 🚀 Como executar o código desta aula
1. Certifique-se de ter o **JDK 17+** instalado.
2. Abra o Eclipse ou sua IDE de preferência.
3. Importe a pasta `aula01`.
4. Execute a classe `Main.java`.
