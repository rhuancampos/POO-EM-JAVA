# ☕ Curso de Java: Programação Orientada a Objetos

Repositório dedicado ao armazenamento de materiais, códigos de exemplo e exercícios práticos das aulas de **Introdução à Programação Orientada a Objetos em Java**.

## 🎯 Propósito do Repositório

Este projeto serve como guia de estudos para alunos que desejam dominar o ecossistema Java. O foco é transitar de conceitos básicos de lógica até a estruturação de aplicações robustas utilizando o paradigma de POO, conectando a teoria com desafios práticos do mercado.

---

## 🛠️ Pré-requisitos

Para rodar os códigos deste repositório, você precisará de:
* **JDK 17** ou superior instalado.
* Uma IDE (recomendamos **Eclipse**, mas pode ser utilizado VS Code ou IntelliJ).
* Git instalado para clonar o projeto.
